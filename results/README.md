# Reproducible results and trace artifacts

`simulation_results.csv` is the 100,000-trial baseline produced by the trace-audited simulator using seed `20260705`.

`traces/` contains deterministic `--simulate-this` transcripts used for manual review. Each state-changing line includes the rule IDs documented in [`../docs/RULES_TRACEABILITY.md`](../docs/RULES_TRACEABILITY.md).

The previous `variant_results.csv` screen was removed because it was produced before the trace-policy rewrite. It must not be reused until variants are regenerated by this engine.

The C++ writer uses a destination lock and atomic replacement. CTest covers the trace matrix and the aggregate smoke run.
